🔬 SCIENTIFIC METHODOLOGY

════════════════════════════════════════════════════════════════════════════════

ALGORITHM: Dixon-Coles Poisson Regression (1997)

The model works by:

1. GOAL INTENSITY ESTIMATION (λ)
   ────────────────────────────────
   λ_home = exp(strength_home + opponent_weakness + home_advantage)
   λ_away = exp(strength_away + opponent_weakness)

   λ (lambda) represents expected goals for each team in a specific matchup.

2. PROBABILITY CALCULATION
   ────────────────────────
   For each possible scoreline (0-0, 1-0, 2-1, etc.), we calculate:
   P(goals_home = i, goals_away = j) = Poisson(i, λ_home) × Poisson(j, λ_away)

   This gives us a full probability matrix for all possible outcomes.

3. DEPENDENCE CORRECTION (ρ)
   ────────────────────────────
   In reality, low-scoring matches (0-0, 1-0, 1-1) occur more often than
   pure Poisson predicts. The rho factor corrects this:
   - P(0-0) is increased (draws more common)
   - P(1-0) and (0-1) are adjusted
   - P(1-1) is handled separately

4. RESULT PROBABILITY AGGREGATION
   ──────────────────────────────
   Home Win (1) = Sum of all P(i,j) where i > j
   Draw (X) = Sum of all P(i,i)
   Away Win (2) = Sum of all P(i,j) where i < j

5. ELO INTEGRATION
   ────────────────
   Team strength parameters are calibrated using historical ELO ratings:
   - Morocco: 1692.7 (Tournament favorite)
   - England: 1671.1
   - Germany: 1643.8
   - France: 1640.2
   - Spain: 1643.1
